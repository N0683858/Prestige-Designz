Position Property

Static

This is the default value. The element is positioned according to the normal flow of the document. The top, right, bottom, left and z-index properties have no effect.

Relative

The element is positioned according to the normal flow of the document, and then offset relative to itself based on the values of top, right, bottom, and left. The offset does not affect the position of any other elements; thus, the space given for the element in the page layout is the same as if position were static. For example, when I set the 'top' and 'right' attribute to 25px, it offset the position of the middle div but the space around it will remain the same.

Absolute

The element is removed from the normal document flow, and no space is created for the element in the page layout. It is positioned relative to its closest positioned ancestor, if any; otherwise, it is placed relative to the initial containing block. Its final position is determined by the values of top, right, bottom, and left. You can see what happens when we set the middle div's position as absolute. You may not be able to see the last div because it is under the middle div since the middle div has been removed from the flow so its not taking up any space.

Fixed

If an element is assigned the property of fixed, then it will be fixed in place. The element is removed from the normal document flow, and no space is created for the element in the page layout. As you can see, the div on the top left stays at the top-left even when we scroll down.

Another property of position which is similar to fixed is sticky. If an element is assigned the sticky position, it will stay in its place, but when you start scrolling, it will remain in that position on your screen. This is ver useful when making a NavBar.

CSS Transitions

Transition property helps transform elements on the webpage. The first square div changes into a circle immediately when you hover over it, almost flickers into a circle. However as you can see the second div slowly transitions into a circle because it has the transition property set to 1 second. The seconds indicate on how quickly you want the transition to happen.

Singling out property

You can single out the property you want the transition effect to take effect on. For example, if you wanted just the background color to take 2s to transition, you can single it out by specifying i.e. transition: background-color 2s. This will only effect the background and the square will flicker into a circle.

Adding Delay

You can add a delay to all properties before they transition or single out each element you want delayed. For example, you can add 1 second delay to the background property and 3 second delay to the border radius property. As you can see the background color changes before the border radius.

Transforming Functions

Transforming functions gives us the ability to animate the elements such as, rotate them, scale and stretch them, translate or skew them across our screens, horizontally or vertically.

Read more on MDN

When you use the transform property, it usually uses the middle of the element as the starting point. However, you can use the 'transform-origin' property to decide where you want center/starting point to be, top right, bottom left etc. The example above shows of a default starting point of a element using the rotate property.
