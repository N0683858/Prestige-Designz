What is Flex Box?

Flexbox is a one-dimensional layout method for laying out items in rows or columns. Flexbox allows us to distribute space dynamically across elements of an unknown size, hence the term 'flex'. It gives us control over how much space each element should take up on the screen.

Flex Direction

In flex box, by default the elements are set on the Y-axis (horizontal), however we can change the direction we want the elements to be displayed.

Row

The 'flex-direction' property allows us to decide which axis direction we want our elements to be displayed. The default flex direction is 'row'. Another of the property we can use to change the direction is called 'row-reverse' which as the name implies, reverses the row of elements.

Column

Another way to display elements inside a flexbox is by displaying them the X-axis (vertically). We can do this by giving the flex-direction property a value of 'column' and 'column-reverse' if we want the columns to be displayed in reverse.

As you can see, the divs elements aren't 200x200 anymore, that's because we set a hight on the container of 500px, so the flexbox makes the div elements smaller to fit inside the container. If we make the container bigger, then the div elements will also grow (maximum they will grow to 200x200).

Justify Content

Flex Start/End

Flex start will push the content to the right, the start of the container, and "flex-end" will push the content to the end of the container.

Center

This will center all the content between both axis.

Space Between/Around

Space between takes all the empty space, and distributes in in between the elements, but not on the outside edges. Space around distributes the empty space equally "around" each element, but it looks a little weird.

Space Evenly

Space evenly ensures all the empty space is distributed evenly between all the elements in the container.

Flex Wrap

When you have too much content inside the container, flexbox will shrink it to make it all fit. However, this will change thw size of the content i.e divs which is not how you intended it to look.

So flexbox has a property called flex-wrap which will push the remaining content onto the next row/column while respecting the original width and size you assigned the content. Or you can use wrap-reverse so the content is wrapped in reverse on the opposite axis..

Align Items

Align items is something we use to distribute our items/content, similar to justify content, however on the Y-axis, whereas justify content distributes space on the main (X-axis).

It also has similar properties such as flex-start, center, flex-end.

One extra value it takes is called baseline which aligns the content using the baseline of our text.

Align Content

Align content is used to distribute empty space between the cross axis only when we have multiple rows/columns. It only takes effect when you have wrap or (wrap reserver) turned on. As you can see in the example below, where are 3 columns and when you assign align-content:space-between it distributes the empty space equally in between the content.

Similarly, if you use the other values such as flex-start it will align the content to the start of the container but the 2 rows still remain.

Align Self

Align self is very similar to align items, except its a property we add to individual items or elements which are not in the flex containers. We can change the alignment along the cross axis for a single element.

Flex-Basis, Grow & Shrink

Flex property is a shorthand for flex-basis, flex-grow and flex-shrink. You can pass in few different values.

Flex-Basis

Flex basis basically sets the initial size of the element you want it to be, ignoring the width or height, depending on what the main axis is. When there is a flex item, it will automatically have flex basis. The default value is set to auto.

Flex-Grow

Flex grow allows you to decide how much empty space an individual element should take up. You may want the first dive to take up twice as more space as the second div. You can do this by using the flex-grow property as it allows elements to grow when there is more space available.

Flex-Shrink

When there is not enough space in the container, flex-shrink determines how much you want an element to shrink compared to other elements in the container. If you give an element the value of 0, it will not shrink.

Media Queries

Media Queries allow us to modify our stylesheets (the way the website should look) depending on certain parameters such as screen size, device type (mobile, tablet etc) or device orientation (portrait/landscape).
